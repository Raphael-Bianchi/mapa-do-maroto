Premissas do Projeto

  • Para que nosso mapa seje o mais certeiro possível, pensei em algumas ideias centrais para guiar nosso trabalho... se não nunca ia conseguir botar em prática kkkkkk

Objetivo

  • A ideia é ter um "resumão" colaborativo. Então qualquer ideia, correção, melhoria ... ou se você já fez um resumo que não temos aqui, é só ir la no GitHub e fazer o pull request!

Premissas

  • Os resumos precisam ser entendíveis e sucintos não apenas uma transcrição das aulas.
  • Linguagem fácil e informal para facilitar o aprendizado. Expressões como “inputar”, “commitar” são suuuuper bem vindas!
  • Vamos tentar nos “limitar” apenas aos conteúdo abordados nas aulas para o resumo não virar um resumão.
  • Tudo compartilhado la no GitHub!
